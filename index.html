Using the NERDS stack

A scalable web app has great performance (as the load increases, the app adjusts itself to handle the increased load), retains a coherent structure as its complexity grows, and is relatively easy for new developers to learn and maintain. I use the NERDS stack to create dynamic and scalable applications.

NODE

Node.js is a JavaScript runtime built on Chrome’s V8 JavaScript engine. Node is designed to build scalable network applications. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js’ package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

Express

Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications. With myriad HTTP utility methods and middleware at your disposal, creating a robust API is quick and easy.

React/Redux

React makes it painless to create interactive UIs. Design simple views for each state in your application, and React will efficiently update and render just the right components when your data changes. Redux brings predictability to your application state by keeping the entire state of the app in a single store.

Databases using SQL

SQL is one of the most popular database languages in use today. Specifically, PostgreSQL is a powerful, open source object-relational database system. It has more than 15 years of active development and a proven architecture that has earned it a strong reputation for reliability, data integrity, and correctness.
