dataLab

DataLab is a desktop application built using Electron. Users can query SQL databases, build live-updating custom charts in D3 based on the results, and then create and share dashboards of charts with other users.

wordScramble

WordScramble is a React Native application I built just for fun. It's a game where users add, subtract, or substitute letters from a given word to create new valid English words. The most exciting part for me was trying to figure out how to use a genetic algorithm to find solutions to the game's puzzles. Genetic algorithms are not the optimal way of finding solutions, but they sure are fun to play with!

Check out the source code.

scentyAnts

ScentyAnts won the Grand Prize at Fullstack Academy's Hackathon. I built it because I was curious about how relatively unsophisticated agents can coordinate their behavior by leaving scent packets for each other. The ants evolve toward equilibrium across 6 factors such as strength health and speed by a combination of recursive descent and a genetic algorithm.
